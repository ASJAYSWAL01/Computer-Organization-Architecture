# 🖥️ Computer Organization & Architecture Practicals

## 📌 About This Repository
This repository contains **Computer Organization & Architecture (COA) practicals** using **Logisim Simulator** along with **.circ circuit files** and **handwritten PDF notes**.

It helps you understand digital logic design and computer architecture concepts in a simple and practical way.

---

## 🧰 Tools Included
🛠️ Logisim Simulator (.exe file included)  
📁 Circuit Files (.circ) for practical implementation  
📄 Handwritten PDF with theory + diagrams
